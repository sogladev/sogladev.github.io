# Lua Automation Toolkit

A collection of utilities and libraries for building automation scripts with Lua.

## Features

- **Fast Execution**: Powered by LuaJIT
- **Simple API**: Easy-to-use interfaces
- **Extensible**: Plugin system for custom functionality
- **Cross-Platform**: Works on all major platforms

## Use Cases

- Build automation
- System administration tasks
- Data processing pipelines
- Game modding and scripting

## Tech Stack

- Lua 5.4 / LuaJIT
- C FFI for performance-critical code
- JSON and YAML parsing
