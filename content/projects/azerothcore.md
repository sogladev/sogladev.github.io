# AzerothCore

AzerothCore is a complete open-source MMORPG framework derived from MaNGOS, TrinityCore, and SunwellCore.

## Features

- **C++ Core**: High-performance server written in modern C++
- **SQL Database**: MySQL-based data storage
- **Modular Design**: Easy to extend with custom modules
- **Active Community**: Large community of developers and users
- **Cross-Platform**: Runs on Linux, Windows, and macOS

## Tech Stack

- C++17
- MySQL 8.0
- CMake
- Boost libraries
