# Rust CLI Tools Collection

Modern command-line tools built with Rust for speed and reliability.

## Included Tools

- **fast-find**: Blazingly fast file search
- **json-query**: Query and transform JSON data
- **log-analyzer**: Parse and analyze log files
- **bench-runner**: Benchmark and profile applications

## Tech Stack

- Rust 1.75+
- Clap for CLI parsing
- Tokio for async operations
- Serde for serialization

## Installation

```bash
cargo install rust-cli-tools
```
